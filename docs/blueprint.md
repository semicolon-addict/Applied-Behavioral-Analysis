# **App Name**: BehaveAsses

## Core Features:

- Role-Based Authentication: Secure authentication and authorization for Admins, Clinicians, and Parents using MongoDB or Firebase.
- Parent Portal: Allows parents to register, fill questionnaires, view child's progress, and communicate with clinicians.
- Clinician Dashboard: Enables clinicians to view assigned children, track progress (ABLLS/AFLLS/DAYC-2), input domain scores, and generate PDF reports using jsPDF.
- Admin Dashboard: Facilitates managing accounts, assigning children, viewing reports, generating analytics, and exporting data (CSV/PDF).
- Progress Visualization: Display progress using charts via Chart.js or Recharts; provide reports for growth tracking with percentage bars and heatmaps.

## Style Guidelines:

- Primary color: Soft Lavender (#E6E6FA) to convey a sense of calm and well-being, suitable for behavioral assessment platforms.
- Background color: Pale Grey (#F5F5F5), nearly desaturated from primary to offer a muted, clean backdrop.
- Accent color: Muted Lilac (#D8BFD8) provides subtle contrast without overwhelming the primary scheme, enhancing key interactive elements.
- Body and headline font: 'Inter' for a modern, objective feel.
- Use role-specific icons with pastel fills on the login page.
- Design clean, card-based layouts to organize assessment details and analytics.
- Incorporate smooth transitions when loading assessment scores to enhance user experience.
